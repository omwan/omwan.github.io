Class Assignments

  • Forbidden Island: Navigate a sinking island to gather all the pieces of a helicopter and bring it to the top of the island before the rising water levels catch you! Made with Java for CS 2510, Spring 2016. Github
  • Kruskal Maze: Creates randomly generated mazes using Kruskal's algorithm for minimum spanning trees; can be solved by manually traversing the maze or triggering a depth-first or breadth-first search algorithm. Made with Java for CS 2510, Spring 2016. Github
  • Tetris: Recreation of classic Tetris game--stack blocks to try to clear complete rows without hitting the top of the screen. Made with Racket for CS 2500, Fall 2015. Github
  • Battleship: Recreation of classic Battleship game against computer AI--try to guess the location of your opponent's ships and sink them all before they find and sink yours! Made with Java (Processing) for Honors CS, Spring 2015. Github | Website
